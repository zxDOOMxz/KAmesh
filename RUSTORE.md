| **Краткое описание** | Общайся без интернета, WiFi и GSM. Сообщения передаются напрямую между телефонами с приложением KAmesh через BLE-сеть. Все данные защищены сквозным шифрованием. |
| **Полное описание** | KAmesh позволяет обмениваться текстовыми и голосовыми сообщениями напрямую между телефонами без интернета, WiFi и сотовой сети. Использует BLE mesh-сеть для ретрансляции сообщений — гарантированная доставка через цепочку устройств. Все сообщения защищены сквозным шифрованием (Signal Protocol).\n\nВозможности:\n- Текстовые сообщения в офлайн-режиме\n- Голосовые сообщения (30 сек, Opus 8 кбит/с)\n- Голосовые вызовы через прямое BLE-соединение\n- Автоматическая ретрансляция через mesh-сеть\n- Сквозное шифрование (X3DH + Double Ratchet)\n- Гарантированная доставка сообщений через mesh-сеть — даже если адресат не в прямой видимости |

## Скриншоты (требуется 2–5 шт.)

Рекомендуемый размер: 1080×1920 px (или 1440×2560).

Сделай скриншоты:
1. **Экран чата** — список сообщений, тёмная тема
2. **Запись голосового сообщения** — кнопка микрофона, таймер
3. **Входящий вызов** — оверлей с кнопками ответа/отклонения
4. **Список контактов** (если реализован) — или экран поиска устройств

## Уведомление ФСБ (ФЗ-533, ст. 12)

Приложение использует собственные средства криптографической защиты информации:
- X3DH (Extended Triple Diffie-Hellman) — асинхронный обмен ключами
- AES-256-GCM — шифрование сообщений
- X25519 — эллиптическая кривая для DH
- Double Ratchet — смена ключей на каждое сообщение

**Перед публикацией в RuStore необходимо подать уведомление в ЦЛСЗ ФСБ России.**

Форма уведомления: https://clsz.fsb.ru (раздел "Уведомления")
Срок рассмотрения: до 30 дней.
